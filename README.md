# ProyectoFamiliar
Sistema de gestión de ferretería de inventario y ventas desarrollado con Flask y SQLite. Permite CRUD de productos, registro de ventas diarias, cálculo de totales y exportación a Excel.

## 🛒 Sistema de Inventario y Ventas

**Aplicación web desarrollada con Python y Flask para gestionar productos, registrar ventas y generar reportes diarios.**

## 🚀 Funcionalidades

- CRUD completo de productos (crear, editar, eliminar)
- Registro de ventas
- Cálculo automático del total vendido por día
- Exportación de ventas a Excel
- Buscador de productos
- Interfaz con plantilla base y estilos personalizados

## 🛠 Tecnologías utilizadas

- Python
- Flask
- SQLite
- Pandas
- OpenPyXL
- HTML
- CSS

# ⚙️ Instalación
`git clone https://github.com/hikariyoungg/ProyyectoFamiliar.git`

### Crear entorno virtual 
`python -m venv venv`
### Activar entorno
`venv\Scripts\activate`
### Instalar dependencias
`pip install -r requirements.txt`

### Ejecutar 
`python app.py`
